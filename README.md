## 🎯 プロジェクト概要

個人事業主から大企業まで、あらゆる規模のビジネスに最適な在庫管理システムのプロダクトサイトです。買い切り型で月額料金なし、3つのプランから選べる柔軟な価格設定が特徴です。

### 主な特徴

- ✅ **買い切り型** - 月額料金なし、一度購入すれば永久に使える
- ✅ **3つのプラン** - ライト（¥980）、スタンダード（¥2,980）、プロ（¥4,980）
- ✅ **マルチデバイス対応** - PC・スマホ・タブレット完全対応
- ✅ **モダンなUI/UX** - 2025年のWebデザイントレンドに準拠
- ✅ **SEO最適化** - 検索エンジンからの集客を最大化
